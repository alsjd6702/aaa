First Work

하이원 리조트

프로젝트 선택 이유 자주 방문하는 사이트이지만, 사용자로서 가장 많이 찾는 내용이 바로 보이지 않아 다소 불편함.
작업 방식 반응형
클론 - 90%, 리뉴얼 - 10%
작업 기간 1달
작업 내용 기존의 웹사이트에서 낮은 가독성과 정적인 분위기를 폰트 속성, 배경색, 주요 콘텐츠 레이아웃 변경과 애니메이션, 마우스 호버 액션 추가로 보완함.

Second Work

르라보

프로젝트 선택 이유 간단해 보이는 구성이지만, 폰트, 디자인, 레이아웃, 콘텐츠만으로 깔끔하면서도 고급스러운 분위기를 나타냄.
작업 방식 서브 페이지 추가 제작
클론 - 100%
작업 기간 2주
작업 내용 비디오 및 이미지 삽입, 마우스 호버 액션을 통한 메뉴 노출 및 메인 이미지 변경, 검색 텍스트 입력 등

Third Work

플래닛 2536

프로젝트 선택 이유 해당 사이트를 방문하여 이용하고자 하였으나, 어플만 있어 PC를 통해 모바일 화면을 보는 것이 불편함.
작업 방식 리뉴얼 - 100%
작업 기간 1달
작업 내용 기존의 사이트에서 반복되는 콘텐츠 배치와 부족한 디자인을 추가로 보완하고자 함. 주요 콘텐츠 구성 및 레이아웃과 디자인 변경, 마우스 호버 액션, 애니메이션 추가로 해당 사이트의 밝고, 활기찬 분위기를 살리고자 함.
